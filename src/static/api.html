Dausport API

Please find below the different URLs to communicate with the API


About bookings:

  • "/" [GET]: get Home page
  • "/bookings" [GET]: get all bookings (per user id)
  • "/" [POST]: json schema {"id": int, "day": str}: Register or unregister for climbing on specified day

About users (signup needed for first auth)

  • "/users" [GET]: Show all users who signed up.
  • "/users/signup" [POST]: json schema {"firstname": str, "lastname": str, "mail": str, "password": str}:
